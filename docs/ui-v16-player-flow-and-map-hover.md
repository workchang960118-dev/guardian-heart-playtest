# UI v16｜玩家流程、地圖 hover／click、教學與大廳收斂

這版整合了以下幾類調整：

1. 地圖互動
- 滑過地格：顯示資訊型 hover 提示
- 點一下地格：開啟互動選單
- 雙擊合法目標地格：快速移動
- 右側地格的浮層強制往左開
- 純說明型 hover 使用 pointer-events-none，降低游標爆閃

2. 流程控制
- 保留「結束回合」在主介面
- 「開始本輪」「處理營火」「AI 下一位」移出主流程，收進房主的流程備援
- 房主端自動化：
  - crisis 階段可自動 start_round
  - campfire 階段在無 blockingWindow 時可自動 resolve_campfire

3. 新手教學
- 改成可關閉、可捲動、多頁籤
- 第一頁直接呈現本局目標與一句世界觀
- 另外補 SR / SP / AP / 壓力、回合玩法、任務與事件說明

4. 大廳遮罩
- 角色指派卡直接顯示能力摘要與起始 SR / SP
- 新增「一鍵 AI 補位」與「清空 AI 補位」

5. 輕動畫
- 壓力變動脈動提示
- 任務完成數脈動提示
- 事件揭示浮層輕脈動
- 輪到你了提示條
- 目前行動中角色 token 脈動

6. 關鍵訊息強調
- 事件條件與未解懲罰數值更醒目
- 任務條件、獎勵與勝利條件中的關鍵詞／數字加強

7. 地圖卡面優化
- 地格名稱移除省略號
- 保持單行顯示
- 角色 token 更醒目
