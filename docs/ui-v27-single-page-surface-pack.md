# UI v27｜單頁主畫面收斂包

本包延續 v26，主修單頁主畫面的資訊密度與操作語言一致性。

## 本輪完成
1. 頂部控制列收斂
- 把教學／目標／世界觀收進「更多」選單。
- 移除玩家主流程中的 AI 下一位入口，避免把房主視為流程推進者。

2. 上方事件條壓薄
- 事件摘要、需求、未解懲罰與快速投入改成更薄的單列樣式。
- 補上「地圖為主，細節看浮窗」提示，強化主頁閱讀邏輯。

3. 任務列／任務浮窗再收斂
- 任務 chip 更薄。
- 浮窗縮成較可讀、較不壓畫面的尺寸。
- 浮窗滑出後直接顯示條件、獎勵、目前狀態與宣告入口。

4. 左側玩家卡瘦身
- 改成較薄的資訊卡。
- SR / SP / AP 改成單列數值 pill。
- 相鄰互助入口改成更貼近玩家卡的細條式快捷操作。

5. 右側手牌區再收斂
- 手牌卡縮薄。
- 右側摘要區改成「選牌後才展開細節」。
- 無牌／未選牌時以簡短提示佔位，不再是大塊工具面板。

6. 觀察者模式分流
- 觀察者不再看到手牌區。
- 改為顯示局面摘要與目前等待節點。

7. 建置補洞
- `/simulation` 頁面補上 Suspense 包裹，避免 production build 在 useSearchParams 上失敗。

## 驗證
- `npx tsc --noEmit`：通過
- `npm run build`：在容器中曾跑到 Next.js page data 階段，但容器程序等待中途斷線；已補上本輪發現的 Suspense 缺口。建議接手後本地再跑一次 build 做最終確認。
