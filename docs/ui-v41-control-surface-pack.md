# UI v41｜Control Surface Pack

本輪延續 v40 的單頁主畫面方向，重點不是新增新流程，而是把「任務列、事件條、單頁指令台」收成更像同一套主控台。

## 本輪完成

### 1. 主控摘要帶（surface status cards）
在任務列下方新增 4 張可直接讀局的摘要卡：
- 事件條
- 任務列
- 單頁指令台
- 目前阻塞

目的：讓玩家不必再先打開進階診斷，就能在主畫面直接知道現在最該先做什麼。

### 2. 單頁指令台再前推
指令台新增：
- 「現在最該先做」主標籤
- 最多 3 條 checklist 式提示
- 快速焦點操作：回到目前地格、查看事件全文、固定任務卡、補齊手牌指定
- 底部說明改為明示目前阻塞與營火佇列狀態

目的：讓單頁指令台更像真正的主流程控制中樞，而不是散落的按鈕集合。

### 3. 右側焦點欄補齊事件與流程層
右側焦點欄新增：
- 事件焦點卡：需求、尚缺值、多人投入門檻、未解懲罰、查看事件全文、帶入常用投入
- 流程阻塞卡：目前卡在哪裡、營火佇列摘要、前兩條營火流程提示、可直接續推營火

保留既有：
- 地圖焦點卡
- 任務焦點卡

目的：把「任務列、事件條、單頁指令台」三者的資訊層級統一，而不是任務特別完整、事件與阻塞只靠零碎提示。
