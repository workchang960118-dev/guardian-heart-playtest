# UI v36｜單頁主畫面再收一刀

這一包延續前幾輪 UI 收斂，繼續朝「玩家主畫面盡量單頁、單視窗完成」的方向補密度與可讀性。

## 本輪完成
1. 左右欄再次縮窄
- 主畫面三欄比例再往中央地圖傾斜。
- 左右資訊欄更薄，地圖主體感再加強。

2. 任務卡更輕
- 任務卡外框與字級再壓低。
- 任務獎勵提示字級縮小，減少搶版面。

3. 玩家卡再瘦身
- 維持 SR / SP / AP 三格快速掃描。
- 標題與 role 行距再收一點。
- 行動中高亮維持，但不再過厚。

4. 右側手牌區更像「輕量操作板」
- 手牌區新增「滑過先看效果，點一下固定選牌」提示。
- 未選牌時補上虛線提示框，避免右欄突然空掉。
- 已選牌區保留快速使用與取消。

5. 回合提示再順一點
- 「輪到你了」提示稍微下移，避免太貼頂部資訊帶。
- 加上半透明背景，辨識度提高但不致太重。

6. 進階摘要區再壓密度
- 隱藏區的戰局摘要／觀察者提示卡片間距與 padding 再縮一點。

## 驗證狀態
- 本容器未提供可直接使用的 node_modules，這一包未完成 build 驗證。
- 建議安裝後優先檢查：
  1. 左右欄是否更不擠地圖
  2. 右側手牌區是否更順手
  3. 任務卡與玩家卡是否更容易掃讀
