# UI refresh v7

本輪目標：讓桌機首屏更接近示意圖的資訊主次，並盡量減少進房後需要往下滑的程度。

## 本輪調整
- 將共同任務列移到主舞台上方，改成更扁平的首屏目標列
- 主舞台欄寬改為 `190 / 1.9fr / 220`，增加中央地圖占比
- 縮小玩家卡與手牌卡密度，讓左右欄更像輔助欄
- 壓縮頂部橫幅與資訊卡高度
- 地圖舞台高度與 tile 尺寸小幅上調，讓中央視覺更突出
- 原本下方重複的共同任務區塊改為 hidden，避免首屏重複與吃高度

## 預期效果
- 首屏更像「左玩家／中地圖／右手牌」的桌面結構
- 共同任務改在地圖上方一眼掃完，不再出現在地圖下方
- 地圖更大，且不需要依賴水平滑動

## 仍待後續觀察
- 小螢幕與筆電高度不足時是否仍需要少量垂直捲動
- 若要再更接近示意圖，可再繼續縮短右側快捷操作高度
