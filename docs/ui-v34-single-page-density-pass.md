# UI v34｜單頁密度補強

這一刀延續 v33 後的主畫面收斂，重點不是再加新系統，而是把單頁 UI 的密度與閱讀順序再壓乾淨。

## 本輪改動

### 1. 主舞台左右欄再縮
- 主畫面三欄比例改為更偏向中央地圖。
- 左右資訊欄寬度再收窄，避免搶掉主舞台。

### 2. 左側玩家卡再瘦身
- 玩家卡改成更緊湊的 3 格 SR / SP / AP 數值區。
- 行動中狀態改成較輕的 amber 高亮。
- 位置與陪伴標記資訊縮成較小的 pill。
- 相鄰互助快捷區塊也一起縮小，減少縱向高度。

### 3. 右側手牌區再收斂
- 手牌預覽卡縮成更薄的卡片。
- 選中的手牌會出現「已選」標記，不靠大面板撐版。
- 右側摘要提示改成更像快速確認，不再是厚重說明塊。

### 4. 任務 rail 文案收斂
- 任務列提示改成更明確的「滑過預覽、點一下固定；任務不再離開主畫面」。
- 保持任務浮窗作為主閱讀／操作入口，不再暗示要切到其他區塊。

## 驗證狀態
- 此容器缺少可用的 Next / React 依賴，未能完成 `next build` 與 `tsc` 驗證。
- 本輪屬於 UI 結構與樣式收斂，建議安裝後直接檢視：
  1. 左右欄是否比前版更不擠地圖
  2. 玩家卡是否更容易一眼掃描
  3. 手牌區是否更像輕量操作面板
