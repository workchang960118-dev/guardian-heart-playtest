# UI v35｜單頁密度與流程提示 follow-up

這一刀延續前面的 UI 收斂，主修幾個仍然會影響內測體感的地方：

## 本輪重點
1. 左右欄再縮
- 三欄比例再偏向中央地圖。
- 左右資訊欄加上更輕的邊框與底色，視覺更像輔助欄位。

2. 左側玩家卡再瘦身
- 玩家卡改成更緊湊的小卡。
- SR / SP / AP 變成三格 pill 化數值區。
- 狀態標籤、位置與陪伴資訊都再縮小。

3. 右側手牌區再收斂
- 手牌預覽卡更薄，並補上「已選」標記。
- 未選牌時加入輕量提示，不再只留空白。
- 已選牌摘要改成更輕量的快速操作區。

4. 結束回合提示加強
- AP = 0 時，「結束回合」按鈕改成更明顯的脈衝樣式。
- 補一句小提示，降低玩家不知道該做什麼的停滯感。

5. 終局結算滾動補強
- 結算頁改成 header 固定、內容區 flex-1 + overflow-y-auto。
- 降低整個結算內容被卡住、看不到下半部的風險。

## 驗證狀態
- 這包是在 handoff 原始碼上繼續做 UI 收斂。
- 目前容器裡沒有完整可用依賴，未重跑 next build / tsc。
- 建議安裝後優先檢查：
  1. 左右欄是否比前版更不擠地圖
  2. 玩家卡與手牌是否更容易掃描
  3. 結算頁是否終於能順利往下捲
