# UI 中文化與用途補充 v1

這次調整的目標是把使用者會直接看到的頁面再往「全中文、可理解」拉近，重點包含：

1. 房間入口頁
   - 英文副標改為中文
   - `join token` 改成 `重進識別碼`
   - `blocking windows` 改成 `棄牌與損失反應視窗`

2. 行為記錄分析頁
   - `action type` 改成 `操作類型`
   - `blocking` 改成 `阻塞視窗`
   - `phase / round / pressure / active` 改成 `階段 / 回合 / 壓力 / 目前輪到`
   - 常見操作類型會顯示中文，並保留原始代碼供比對

3. AI 模擬比較報表頁
   - 頁首英文副標改成中文用途說明
   - `fit score / baseline / confidence / gap score / impact` 等詞改成中文
   - `Round` 改成 `回合`
   - 補上多處「這頁在幹嘛」的中文提示

4. 房間主畫面中的明顯英文
   - `Claude 地圖 v1` 改為 `地圖版型 v1`
   - `blocking window` 改為 `阻塞視窗`

這一刀是介面文字層與說明層調整，不改規則與流程。
