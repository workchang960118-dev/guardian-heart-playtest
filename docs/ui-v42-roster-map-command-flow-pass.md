# UI v42｜玩家列 × 地圖 × 指令台動作串接

本輪將主畫面再往「單頁操作台」推一步，重點不是新增新系統，而是把既有三塊 UI 真正接成一條操作流：

1. 玩家列可直接點選隊友作為焦點，並同步帶入主畫面流程。
2. 地圖地格浮層若有同格玩家，可直接點玩家膠囊，把焦點帶回玩家列與指令台。
3. 單頁指令台新增「隊友焦點」區，顯示位置、關係與可直接執行的快捷動作。

本輪主要 UI 變更：
- 玩家卡新增「焦點」視覺狀態，並支援鍵盤 Enter/Space 選取。
- 點玩家卡會同步：互助對象、必要時的手牌隊友目標，以及該玩家所在的地圖焦點。
- 地圖焦點浮層與詳細區，若同格有玩家，現在可直接點選該玩家作為隊友焦點。
- 單頁指令台新增隊友焦點卡，提供：看他所在格、帶入互助對象、帶入手牌隊友、直接互助目前選定資源。
