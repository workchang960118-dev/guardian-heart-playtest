# UI v37｜玩家卡可讀性、手牌單頁直出、任務列狀態帶

## 這輪處理範圍
這輪仍鎖在主畫面 UI，不回去碰分析腳本或非 UI 主線。重點是把「單頁能做的事」再往前推，並順手修掉幾個閱讀密度問題。

## 本輪完成

### 1. 左側玩家卡可讀性收斂
- 角色名稱改成可兩行顯示，不再因為「行動中 / 你 / AI」標籤擠壓而容易斷得很難看。
- 右上狀態標籤改成較固定的窄欄，避免把角色名稱硬擠成怪異雙行。
- 玩家卡內的相鄰互助入口改成更明確的「玩家卡快捷」語氣，按鈕也從小圓 pill 收成更穩定的雙欄按鈕。

### 2. 手牌互動往單頁再推一步
- 每張手牌現在會直接標示：
  - `可直出`：不需指定目標
  - `指定隊友 / 指定地格 / 指定資源 / 指定隊友投入`：需要補的欄位
- 選中手牌後，右側／上側的「目前選擇」區會直接顯示需求 badge。
- 若該牌 **不需任何額外指定**，現在可直接在主畫面按下 **「直接使用」**，不必再打開操作 modal。
- 若仍需指定目標，則保留 **「展開操作」**，並明示還缺哪些欄位。

### 3. 任務列補一條即時狀態帶
- 任務 rail 右上提示不再只是固定說明字，而會優先顯示目前焦點任務的名稱與狀態。
- chips 下方新增一條緊湊狀態帶，會顯示：
  - 目前焦點任務
  - 狀態標籤
  - 一句 summary
- 這讓玩家即使不打開完整浮窗，也能先快速知道現在是哪張任務、卡在哪裡。
