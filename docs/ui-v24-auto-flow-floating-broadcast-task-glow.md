# UI v24｜自動推進＋無背景中央廣播＋任務亮起

## 這刀主修
1. **中央廣播改成無背景浮字**
   - 不再切出獨立區塊
   - 主畫面中央偏上，白字、放大、1–3 秒後淡出
   - 「輪到你了」也走同樣的浮字邏輯

2. **本輪開始／進入營火改自動推進**
   - 危機階段會自動 `start_round`
   - 所有玩家結束回合進 campfire 後，會自動 `resolve_campfire`
   - 任務窗口若沒有可宣告任務，會自動 `advance_campfire`
   - 不再把 host 當流程必要節點；由一個一般玩家端自動驅動，其他人只看同步結果

3. **黑夜過場改成全黑轉場**
   - 營火結束後顯示全黑過場
   - 顯示「夜色又深了一層……」與剩餘天數／壓力變化
   - 約 1.5 秒後自動接到事件揭示

4. **任務列改成 hover 可看、達成會亮、營火可宣告**
   - 每張任務 chip 都可 hover 看條件／獎勵／狀態
   - 條件已滿但尚未到營火：顯示 `待營火宣告`
   - 營火中可宣告：顯示 `可宣告` 並亮起
   - 右上補 `宣告完成` 按鈕
   - 若這輪不想宣告，也可按 `不宣告，繼續營火`

## 目前口徑
- 任務仍維持 **營火階段主動宣告**，沒有前移到玩家回合即時完成
- 流程按鈕只保留真正需要的：
  - `開始遊戲`
  - `AI 下一位`（僅備援）
- `開始本輪`、`開始營火` 已退出主流程按鈕區
