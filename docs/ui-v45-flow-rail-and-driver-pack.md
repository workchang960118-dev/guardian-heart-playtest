# UI v45｜Flow rail + driver pack

本輪延續 v44 的 control cockpit 方向，但再把主畫面往「單頁主控台」推一層，重點不是新增新功能，而是把既有事件／任務／能力／營火資訊拉到更同一條操作線上。

## 本輪完成

### 1. 主流程導引列（Flow rail）
- 在主畫面摘要卡下方新增一層主流程導引列。
- 將事件、任務、能力、營火四條線用同一組 compact 卡呈現。
- 視狀態分為：
  - 完成／可續推：綠色
  - 目前焦點／活躍：紫色
  - 尚未處理或待進入：石色
- 目的：讓玩家不必跳到右側焦點欄，也能先看出整局現在卡在哪條線上。

### 2. 單頁指令台補上三張快看卡
- 在「目前卡住原因」區塊後新增三張 compact 快看卡：
  - 事件快看
  - 任務快看
  - 能力快看
- 三張卡都直接吃現有診斷資料，不另開新規則。
- 目的是縮短「看到卡住 → 找到右欄 → 才知道要補哪裡」的距離。

### 3. 手機底部操作帶再補三張 driver 卡
- 在原本按鈕列與兩張狀態卡下方，再新增三張 compact driver 卡：
  - 事件
  - 任務
  - 營火
- 讓手機版不只看到按鈕，還能直接看出：
  - 事件還缺多少
  - 任務能不能宣告
  - 營火目前卡在哪

## 設計意圖
- 讓主畫面更像「操作主控台」而不是「很多面板並排」。
- 優先降低：
  - 玩家不知道現在先做什麼
  - 任務／事件／能力資訊散在不同區塊
  - 手機版只有操作沒有診斷
