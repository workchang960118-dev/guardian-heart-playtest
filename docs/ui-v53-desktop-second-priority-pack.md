# UI v53｜桌機第二優先整包收斂

本輪集中處理第二優先：
1. 營火／blocking flow 再清楚
2. 任務宣告體驗再補
3. 事件全文／手牌全文之外，再把主畫面與右欄補成更容易檢查與宣告的桌機控制面

## 本輪完成

### 一、桌機主畫面新增兩塊矩陣
- 新增「任務宣告窗口 / 損失與棄牌窗口」雙矩陣
- 新增「任務宣告矩陣 / 營火阻塞矩陣」雙矩陣
- 讓桌機版不只顯示焦點，而是直接告訴玩家：
  - 現在卡在哪
  - 哪種阻塞正在發生
  - 目前有幾張任務可宣告、接近完成、仍待補條件

### 二、右欄主控卡補 checklist
- 事件主控
- 任務主控
- 手牌主控
- 流程主控

四張主控卡現在都會帶出前兩條最重要的 checklist / 下一步提示，降低桌機右欄只剩摘要一句話的問題。

### 三、任務宣告體驗補成可讀矩陣
- 直接統計：可宣告 / 接近完成 / 待補條件 / 已完成
- 讓桌機版能快速判斷：
  - 現在值不值得開任務全文
  - 是否已到營火宣告窗口
  - 還是應該先補條件再回來

### 四、營火與 blocking 再清楚
- 損失反應、棄牌阻塞、營火可續推狀態現在都有矩陣摘要
- 比前版更容易看懂：
  - 目前是 loss / discard / task window 哪一種卡住
  - 後面是否還有待處理佇列
  - 能否直接續推營火
