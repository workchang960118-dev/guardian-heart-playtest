# UI v33 follow-up polish

這包延續 v32，再往前收幾個容易一起動的 UI 細節：

1. 危急示警與中央廣播分層更明確
   - 危急示警改停在右上獨立區塊，不再與中央廣播共用同一個視覺中心。

2. 任務浮窗再穩定
   - 桌機任務浮窗加強層次與可讀性。
   - 手機任務浮窗改成帶半透明背景的底部浮窗，並限制高度、可捲動，避免內容被螢幕吃掉。
   - 桌機浮窗定位預估高度上修，減少靠近視窗邊緣時被截斷。

3. 結束回合提示加強
   - AP 歸零時，結束回合按鈕改成更明顯的脈衝樣式。

4. 終局／結算畫面可捲動性再補強
   - 內容區明確改成 flex-1 + overflow-y-auto，避免整個結算視窗看得到卻無法捲動。

5. Overlay scroll 餘量放大
   - 通用 overlay 最大高度提高，降低內容卡住機率。
