# UI v32｜提示拆層與任務浮窗收斂

本包主修：

1. 將危急示警從中央廣播拆出，改成固定於畫面上方的獨立紅色警示，不再佔用中央廣播位置。
2. 中央廣播改成更深色、仍維持無背景、置中懸浮。
3. 任務浮窗補成雙模式：
   - 桌機：任務列附近 portal 浮窗
   - 手機：底部浮窗
4. 任務浮窗可直接顯示條件、獎勵、狀態與阻塞原因；營火可宣告時直接提供按鈕。

6. AP 歸零時，結束回合按鈕下補一句明確提示。
7. 黑夜過場節奏再放慢，讓漸暗／漸亮更明顯。
