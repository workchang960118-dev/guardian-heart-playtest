# 守護之心｜重開對話前情提要 v1

## 一、目前最高口徑
- 目標不是先追漂亮，而是做出**接近成品、可近、舒服、能內測**的 UI。
- 玩家主畫面以 **單頁／單視窗完成** 為主。
- 觀察者模式／開發者模式才放更多 log、AI、debug 資訊。

## 二、目前 UI 主線已成立的部分
1. 中央地圖已成為主體。
2. 上方已有流程控制＋事件／戰況條雛形。
3. 任務列已改成橫向可滑動，不再全部硬擠。
4. 大廳已改成遮罩式準備介面。
5. 世界觀與本局目標已有入口。
6. 壓力已改成水平條雛形。

## 三、目前最重要的已知缺口
1. 某些 overlay / modal 仍會卡住，內容不能往下拉。
2. 點手牌後的互動還不夠完整。
3. 相鄰互助還沒真正貼近地圖／玩家 token。
4. 事件揭示、黑夜過場、單行動作廣播還沒完整形成節奏。
5. 新手教學還不是玩家可選、漸進式版本。

## 四、目前已拍板的設計判斷
### 1. 行動卡與反應
- 不是所有反應型行動卡都能在非自己回合即時回應。
- 一般行動卡仍以自己回合、1AP 為主。
- 陪伴標記才是目前最重的跨時機救援。
- 未來若要有反應牌，應該是少數、明文標示、不是通則。

### 2. 主要互動方向
- 地格：hover 看資訊，click 開操作，double click 做熟手快捷。
- 手牌：hover 看效果，click 開操作。
- 任務：任務列直接宣告。
- 陪伴標記：只在損失反應時出現，不做常駐按鈕。

### 3. 流程演出方向
- 大廳 → 開場導入（世界觀＋勝利目標）→ 第一張事件。
- 營火後：畫面漸暗，顯示「夜色又深了一層……」，再更新剩餘天數與壓力，接著揭示新事件。
- 中央可做不擋閱讀的單行透明動作廣播。

## 五、這包先修到的部分
1. 部分 modal 已改成可捲動。
2. 地格選取不再預設黏住目前地格；可取消選取。
3. 右上手牌區補了第一層 click 操作。
4. 事件投入人數 hover 會顯示投入玩家名單或無人投入。
