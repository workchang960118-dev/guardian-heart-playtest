# UI refresh v6

## 這輪修正
1. 修正手牌重複 cardId 造成的 React duplicate key 警告。
2. 將房間頁上方橫幅改成三段式：房間資訊／事件摘要／核心指標與操作按鈕。
3. 將「快速推進」移回首屏右側，避免開始遊戲仍要往下找。
4. 收掉首屏下方重複的事件／快速推進大區塊，降低首屏高度。
5. 地圖區改成不靠水平捲動，並把中央主舞台再放大。
6. 地圖細節、閱讀提示與快捷操作改收進可折疊區，避免主畫面太工程感。

## 本輪目的
- 修掉會阻礙操作的 duplicate key 問題。
- 讓房間頁更接近最初示意圖的資訊主次：
  - 上方：房號 / 階段 / 事件 / 核心指標 / 主按鈕
  - 中間：左玩家 / 中地圖 / 右手牌與大廳操作
  - 下方：共同任務列
- 讓中央地圖區更大，且減少非必要說明佔據首屏。
