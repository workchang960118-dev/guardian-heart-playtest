# UI v20｜流程演出節奏統一

本包主修：事件揭示／黑夜過場／損失反應三種窗口的節奏統一。

## 本輪調整
1. 黑夜過場改為手動確認，不再自動快速消失。
2. 若黑夜過場後緊接事件揭示，改為由玩家按下「看下一張事件」再進入事件視窗。
3. 事件揭示 footer 明確提示：不自動關閉，先閱讀再確認。
4. 損失反應視窗改用與其他 overlay 相同的 `RoomOverlayDialog` 結構。
5. 損失反應視窗改為不自動消失，明確等待陪伴標記或房主確認。
6. 「輪到你了」提示停留時間拉長。

## 目的
- 減少流程演出過快造成的資訊滑過。
- 讓玩家在事件／夜晚／損失三個關鍵節點都有一致的閱讀與思考節奏。
- 保持單頁主畫面的同時，避免阻塞視窗的規則感與提示感不一致。
