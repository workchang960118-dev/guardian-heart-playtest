# UI refresh v9

這輪是接在 v8 task rail 後的同層 UI 收斂，不動規則與資料流。

## 本輪調整
1. 上方橫幅再靠近「戰局摘要」語言
   - 左：房號 / 階段 / 身分
   - 中：事件名稱 / 解決條件 / 未解懲罰 / 尚缺資源
   - 右：回合 / 壓力 / 任務目標 / 輪到誰 / 主要連結

2. 任務列維持「預設 5 張寬度，超出左右滑動」
   - 卡片高度再壓低
   - 任務列高度再下降，減少首屏被往下擠

3. 中央地圖區再放大
   - 主舞台欄寬調整成更偏中間吃重
   - 地圖容器高度提高
   - 地格按鈕尺寸提高，讓中央地圖更像主角

4. 左右欄再瘦身
   - 玩家卡改得更扁
   - 手牌卡改得更緊湊
   - 右側流程控制區縮短，不再像工程管理面板

## 這輪目標
- 桌機首屏更接近「少滑動」
- 戰局資訊更像桌面戰況摘要
- 中央地圖的視覺主體更明確
