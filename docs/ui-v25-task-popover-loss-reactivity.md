# UI v25｜任務懸浮視窗＋即時反應分級

## 這刀修什麼

### 1. 任務列改成真正可讀的懸浮視窗
- 上方任務 chip 現在支援 hover / focus / click。
- 會在任務列下方浮出可讀浮窗，不再只是黑色小 tooltip。
- 浮窗內直接顯示：
  - 任務名稱
  - 規則文字
  - 達成條件
  - 獎勵
  - 目前狀態
  - 阻塞原因
- 若目前可宣告，浮窗內可直接按「宣告完成」。
- 若正在營火任務窗，也可直接按「不宣告，繼續營火」。

### 2. 即時反應改成輕重分級
- 不再所有損失都用整個全屏大窗阻擋畫面。
- **一般損失**：改用底部中央的輕量即時反應條。
  - 可看到損失來源、SR/SP 變化。
  - 若可用陪伴標記，可直接在反應條上操作。
  - 若無人回應，可由任一玩家直接套用。
- **重損／可能歸零**：仍保留完整阻塞視窗。
  - 條件目前採：
    - 這筆損失可能讓 SR 歸零，或
    - 這筆損失可能讓 SP 歸零，或
    - 總損失量達 2 以上。

## 目的
- 任務資訊要在單頁內可讀，不靠點開大面板，也不靠難抓的小 tooltip。
- 即時反應維持規則正確，但避免一般損失也把整個畫面打斷。
